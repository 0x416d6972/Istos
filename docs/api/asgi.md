# ASGI API

Run an Istos mesh inside FastAPI or Starlette. The ASGI server owns the HTTP
port; Istos rides the lifespan (`serving()` with HTTP off).

Details: [HTTP Gateway — co-hosting](../user-guide/http-gateway.md).
